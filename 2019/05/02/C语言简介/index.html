C语言简介

C语言

C语言是一门编译型语言，广泛用于系统软件开发和应用软件开发。1969~1973年，Dennis Ritchie和Ken Thompson在贝尔实验室共同设计、开发了C语言。C语言被设计的初衷是为了移植与开发UNIX操作系统，而随着UNIX的流行，C语言也逐渐开始成为通用的程序设计语言。

标准化历程

1978年，Dennis Ritchie和Brian Kernighan合作出版了《C程序设计语言》第一版，书中所介绍的语言标准被称为“K&R C”；

1989年，美国国家标准协会（ANSI）对C语言进行了标准化，此标准命名为C89；

1990年，国际标准化组织（ISO）规定了C语言的国际标准C90；

1999年，ISO发布了C99；

2011年ISO又发布了C11，但貌似许多条标准并未被实现；

2018年，C18发布，它对C11的内容进行了补充与修正。

C的主要特性

  • 它是一种结构化的过程式编程语言
  • 属于静态、强类型语言
  • 拥有封装多种类型变量的结构体（结构体可以说是「面向对象编程」中对象的前身）
  • 可以操纵内存的强大功能——指针
  • 预处理——令C的编译更具有弹性

C作为入门语言的意义

C作为计算机领域的初代语言，它的设计仅由2人完成。这意味着C拥有简洁的语法，且不存在许多繁杂的特性（俗称“坑”），所以想要在短时间内快速知晓编程语言的基础概念，从C入门最适合不过了。不无联系的是，C之后的许多新语言都是借鉴C设计的，于是它们的语法大都和C具有相似之处，因此学完C后再去学其他语言可以减少你的学习周期。

学计算机的人，如果要让自己的水平更进一步，就必须接触算法与数据结构。其中，用C/C++来学数据结构的效果最为全面，因为C能带给学习者对于底层原理的启示,学成C再接触其他语言的数据结构很容易进行切换。

编程社还为社员规划好了第二阶段的学习。「深入学习Linux」也是我们的选项之一，届时必定会涉及到C的系统级编程，学习者可借此机会继续深入学习C语言。C还是最适合用于深入计算机系统底层的高级语言，计算机学习者进阶必读的《深入理解计算机系统》中的大量实例都是用C写的（有少量x86汇编语言写的实例）。

我们不会在第一阶段就把C学得太深入，毕竟我们最多只有1年时间来完成第一阶段。

C语言基础教材购买：《C Primer Plus》

GNU项目

这是由Richard Stallman于1984年发起的一个免税慈善项目，它的目标非常宏大：开发出一个完整的类UNIX系统，其源代码能够不受限制地修改和传播。GNU项目已经开发出了一个包含UNIX操作系统的所有主要部件的环境，但内核除外，内核是由Linux项目独立发展而来的。

现代开放源代码运动的思想起源是GNU项目中自由软件的概念。而且GNU工具还为Linux内核提供了环境，使得Linux能像如今这般大受欢迎。总之，GNU项目取得了非凡的成绩。
